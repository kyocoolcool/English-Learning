# Simple Future

### Purpose

1. 對未來做預測 will / be going to
2. 牽涉到意願的話,使用 will
3. 事先計劃好的,使用 be going to, 利用進行式代替未來
4. 當牽涉到時間副詞子句,可用現在簡單代替未來

{% hint style="info" %}
**should** 用在疑問句,並且主詞是 I, We 表示徵求對方同意
{% endhint %}
